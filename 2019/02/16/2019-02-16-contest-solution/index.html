T1

思路

我们设$d_i=(i \times A + B) mod N$。
对于一次询问:

  • 当$T_i=0$时,$ans+=[d_{p+i} \ge C]$
  • 当$T_i=1$时,$ans+=[d_{p+i} < C]$

因为$d_{p+i}=d_p+A\times i$,式子可以化为

  • 当$T_i=0$时,$ans+=[(d_p+i\times A) \mod N \ge C]$
  • 当$T_i=1$时,$ans+=[(d_p+i\times A) \mod N < C]$

可以发现,当$d_p$取某(1或2)个区间时,$(d_p+i\times A) \mod N \ge C$是成立的。

更明确地:

  1. 当$d_p>C$时,$d_p\in [0,N - p - 1]\text{或}d_p\in [N - p + C, N - 1]$时$(d_p+i\times A) \mod N \ge C$成立
  2. 当$d_p\le C$时,$d_p\in [C - p, N - p - 1]$时$(d_p+i\times A) \mod N \ge C$成立

很明显当$d_p$取其他区间内的数时$< C$成立。

然后就可以用区间开点线段树维护区间内$i\times A \mod N$的出现次数,然后区间查询上述区间就好了。

T2

思路

我们用两个栈来模拟双端队列。
由于MOD比较小,对于每个栈,我们可以记录每个元素到栈底的答案。可以用背包$O(MOD)$转移。
删除元素直接删除栈顶元素即可。
如果要删除的栈为空,我们暴力重新分配两个栈中的元素。每个栈分一半。复杂度$O(NlogN\times MOD)$(懒得分析
查询时可以用单调队列或ST表。

T3

思路

看着很像虚树呀。。。
实际上不用完整地套虚树模板。
黈力的位置记为$t(ou)l(i)$
先考虑点$v(v \neq tl)$。
如果只有$tl$,$v$两个点,肯定有一条边(记为分割边),割开这条边,能把树分成”属于tl”和”属于v”两部分。
如果有其他点,我们可以发现如果某条分割边至$tl$的路径有其他分割边,该分割边是无效的。因为事实上,$tl$不会扩展到该分割边,到前面的分割边就已经停止扩展。

把每个有玩家的点按照DFS序排序。一开始res为N。
枚举每个点$v$,分两种情况讨论。

  1. $tl$与$v$的分割边不在$tl$至$root$的路径上,如果该分割边有效,减去$size[v]$
  2. $tl$与$v$的分割边在$tl$至$root$的路径上,如果该分割边有效,减去$N-size[v]$

比较麻烦的就是如何判断分割边是否有效。
可以先判断情况2。找出最近的在$tl$至$root$的路径上的分割边$(a,b)(dep[a]>dep[b])$即可。
很明显,如果一条分割边不在以$a$为根的子树中,该分割边一定是无效的。
不然的话,可以将分割边离$tl$较远的点按照DFS序排序,减去某个点的size后过滤掉其子树中的点即可。

由于我最开始将$v$在$tl$子树分作第三种情况,代码略显冗长,可以写得更加简短。
